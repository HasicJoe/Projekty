Implementace

  • iis_package/__init__.py - Vytvorenie aplikácie, vytvorenie databázového spojenia
  • iis_package/administrator.py - Logika pre administrátora
  • iis_package/auth.py - Registrácia, prihlasovanie, odhlasovanie, generovanie navbaru
  • iis_package/generator.py - Generovanie dát pre databázu
  • iis_package/models.py - Databázové modely
  • iis_package/operator.py - Logika pre dopravcu
  • iis_package/passenger.py - Logika pre registrovaných pasažierov
  • iis_package/personnel.py - Logika pre personál.
  • iis_package/views.py - Logika dostupná pre všetkych uživateľov systému
  • iis_package/templates/* - HTML a Javascript šablóny

Databáze

Instalace Linux

  • sudo apt install mysql-server
  • sudo mysql_secure_installation
  • sudo apt-get install python3-dev default-libmysqlclient-dev build-essential (Debian, Ubuntu)
  • sudo yum install python3-devel mysql-devel (RHEL, Centos)
  • pip install -r requirements.txt
  • ./run - skript pre lokálne spustenie aplikácie
  • http://localhost:5000/gen_data - skript pre inicializáciu dát v db
Pre spustenie na platforme Windows treba použiť nasledujúce príkazy:
  • export FLASK_APP=main
  • flask run
